# nvcd
Simple plugin to help you more easily change the Neovim working directory.

## Getting Started

### Required Dependencies 
None

### Installation 

Using packer.nvim
```
    use ('j--c/nvcd')
```
Using vim-plug
```
    Plug 'j--c/nvcd'
```
Using dein
```
    call dein#add('nvim-lua/plenary.nvim')
```
